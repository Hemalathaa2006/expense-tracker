# Personal Expense Tracker

This project is a simple Personal Expense Tracker built using Python and Streamlit.

## Features
- Add daily expenses
- Categorize spending
- Visualize expenses with charts
- Simple UI

## Technologies Used
- Python
- Streamlit
- Pandas
- Matplotlib

## Run the Project

Install libraries:

pip install -r requirements.txt

Run the application:

streamlit run app.py
